# Draw a flappybird

We’re making a new project and start by deleting the cat.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb1FDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--0544b08fb75c383948423b4f9317dac087c0e357/Screenshot%202025-09-05%20194059.png)

Then we click the Paint/Draw button.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb1VDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--e12f585a60bc487d962f522a5f9cec0fb58807fa/Screenshot%202025-09-05%20194109.png)

Now we’re in drawing mode for a new sprite, where we select the circle tool.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb1lDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--239a697013dde4051d59f4a0cb0c2c48c19298e4/Screenshot%202025-09-05%20194133.png)

You can pick another color by clicking Fill and adjusting the sliders.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb2NDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--b4ba35c8da2708dd04ae0f09685f06eb147183a3/Screenshot%202025-09-05%20194153.png)

A flappy bird is really just made of a bunch of circles, so start by drawing its body with a long purple oval.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb2dDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--6319d74a76f811a28cf397db9158c16e8382c3be/Screenshot%202025-09-05%20194210.png)

Then make a white circle for the wing and another white circle for the eye.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb2tDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--e03bfbcd61c1e2c68e879f72d3852117c685b459/Screenshot%202025-09-05%20194310.png)

Use the brush tool to make a dot in the eye.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb29DIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--44ef4142d0f0855ce6ac67655ec46eb0b208a20c/Screenshot%202025-09-05%20194319.png)

Make an orange circle for the beak.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb3NDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--8a18808828ec1ab90d5de1918b76abf0d7bc60ca/Screenshot%202025-09-05%20194356.png)

Notice that we are in the costumes tab. You can make code by clicking the other tab.

![](https://guides.codingpirates.dk/rails/active_storage/blobs/redirect/eyJfcmFpbHMiOnsibWVzc2FnZSI6IkJBaHBBb3dDIiwiZXhwIjpudWxsLCJwdXIiOiJibG9iX2lkIn19--9850246bf60566ed578c049483ff199a46262d6e/Screenshot%202025-09-05%20194408.png)
