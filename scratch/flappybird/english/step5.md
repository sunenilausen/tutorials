# Lose when hitting obstacle

Make sure our sprites are named so it makes sense. You can change the name by typing in the "Sprite" field.

![](images/Screenshot%202025-09-05%20201113.png)

Start a new code sequence on the obstacle sprite!
Use a "forever," an "if," and then use "touching" and select pipe.
If true, then "stop all."

![](images/Screenshot%202025-09-05%20201151.png)

The basic elements of Flappy Bird are now complete! We can play the game.
